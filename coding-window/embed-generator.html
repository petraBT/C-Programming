Canvas Embed Generator

Paste a one-off starting point below to get iframe HTML for Canvas — nothing gets hosted or saved anywhere, everything lives in the URL itself.

Option A — Inline (iframe)

Embedded directly in the Canvas page. Canvas doesn't isolate its own pages, so this always falls back to a "type your input first" box instead of a live terminal — fine for exercises with little or no stdin, otherwise use Option B.

Option B — Opens in a new tab (fully interactive)

A plain link instead of an iframe. Opened this way, the exercise is its own top-level tab with no parent frame in the way, so it gets real interactive typing - same as the book. Trade-off: it opens outside the Canvas page rather than sitting inline in it.
